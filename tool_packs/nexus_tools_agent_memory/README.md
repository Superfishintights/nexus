# Nexus tool pack: agent memory

HTTP client tools for the `agent-memory` service.

Add this package root to `NEXUS_TOOL_PACKAGES`:

```bash
export NEXUS_TOOL_PACKAGES="nexus_tools_agent_memory"
```

Required settings:

- `AGENT_MEMORY_BASE_URL`
- `AGENT_MEMORY_TOKEN`

Optional settings:

- `AGENT_MEMORY_DEFAULT_PROFILE`
- `AGENT_MEMORY_AGENT_ID`
- `AGENT_MEMORY_TIMEOUT_S`
